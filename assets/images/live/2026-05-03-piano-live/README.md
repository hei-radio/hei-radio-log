# 2026-05-03 피아노 라이브 이미지

피아노 라이브 방송과 후기 게시글에 쓰기 위해 만든 공개용 이미지 결과물입니다.

## 생성 흐름

- `room-no-person-01.png`
  - 처음 만든 조용한 피아노 방송 이미지.
  - 사람이 없어 조금 무섭게 느껴질 수 있다는 피드백을 받았다.
- `room-long-haired-person-01.png`
  - 사람을 넣어달라고 요청한 뒤 나온 변형.
  - 긴 머리 인물처럼 보여 원하는 이미지와는 조금 달랐다.
- `wide-older-person-01.png`
  - 남성 인물로 수정해달라고 요청한 뒤 나온 넓은 비율 변형.
  - 나이대가 더 높아 보이는 느낌이 있었다.
- `wide-30s-male-01.png`
  - 30대 남성 느낌으로 다시 요청해 나온 넓은 비율 변형.
- `instagram-post-01.png`
  - 인스타그램 게시글용으로 만든 피아노 방송 후기 이미지.
- `spoon-post-recommended-songs-01.png`
  - 청취자분들이 추천해준 피아노 연습곡을 정리해 스푼 포스트에 올린 이미지.
- `generation-grid-01.png`
  - 여러 변형을 한 번에 모아둔 생성 결과 모음.

## 사용 메모

- 스푼에서는 목소리, 이미지, 브금이 청취자에게 보이는 주요 분위기 요소다.
- 실제 사람이나 청취자 정보가 들어간 이미지는 사용하지 않는다.
- 악보 원본이나 저작권 자료를 식별할 수 있는 이미지는 공개 저장소에 올리지 않는다.
