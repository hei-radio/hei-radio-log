# 피아노 연습 기록

피아노 방송에서 추천받은 곡과 연습 과정을 정리합니다. 이 폴더에는 공개해도 되는 연습 기록, 어려운 구간 메모, 짧은 회고만 남깁니다.

악보 원본, MR, 전체 연주 녹음, 전체 연주 영상은 공개 저장소에 올리지 않습니다. 필요한 파일은 로컬 비공개 폴더나 개인 클라우드에 따로 보관하고, 이 저장소에는 파일명이나 공유 링크도 남기지 않습니다.

## 곡별 기록 방식

각 곡은 아래 네 가지를 분리해서 기록합니다.

- 연습 구간: 오늘 실제로 반복한 마디, 파트, 손 위치, 리듬
- 메모: 헷갈린 부분, 손이 꼬이는 부분, 다음에 확인할 점
- 녹음: 공개 파일 대신 날짜별 녹음 여부와 들으며 느낀 점만 기록
- 회고: 오늘 연습이 어땠는지, 다음 방송에서 어떻게 다룰지

## 공개 저장소에 남기는 것

- 곡 제목
- 연습한 구간의 간단한 설명
- 어려웠던 점
- 연습 후 느낀 점
- 다음 연습 계획

## 공개 저장소에 남기지 않는 것

- 악보 원본 이미지, PDF, MusicXML, MIDI
- MR, 음원, 전체 연주 녹음, 전체 연주 영상
- 악보를 받은 경로, 개인 이메일, 공유 링크
- 유료 자료를 재현할 수 있을 만큼 자세한 악보 설명
