# Image Assets

방송 후기, 인스타그램 게시글, 유튜브 영상 등에 사용할 공개용 이미지 결과물을 보관합니다.

## 저장 기준

- AI로 생성한 최종 이미지 결과물만 저장합니다.
- 프롬프트 원문, 생성 과정 캡처, 개인 대화 캡처는 저장하지 않습니다.
- 방송자, 청취자, 계정 정보, 이메일, 링크, QR 코드가 보이는 이미지는 저장하지 않습니다.
- 악보 원본, 음원 화면, 저작권 자료를 식별할 수 있는 이미지는 저장하지 않습니다.

## 권장 구조

```text
assets/
  images/
    live/
      YYYY-MM-DD-topic/
        cover-01.png
        story-01.png
```

## 파일명 예시

```text
2026-05-03-piano-live-cover-01.png
2026-05-03-piano-live-story-01.png
```
