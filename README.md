# Hei Radio Logs 🎙️

퇴근 후 조용히 마이크 앞에 앉아 남기는 방송 기록이자 성장 기록입니다.

이 저장소는 스푼 라디오에서 라이브 방송을 하거나 다른 방송을 탐방하며 느낀 점을 차분히 정리하기 위해 만들었습니다. 하루의 소음이 조금 가라앉은 뒤, 어떤 분위기가 좋았는지, 어떤 말투가 편안했는지, 다음에는 무엇을 다르게 해보고 싶은지를 남깁니다.

기록의 중심은 사람의 정보가 아니라 방송의 감정과 흐름입니다. 회사명, 실명, 지역, 특정 커뮤니티명, 프로젝트명, 청취자 닉네임 원문은 적지 않습니다. 대신 익명 표현과 넓은 범주의 단어를 사용합니다.

## 기록 원칙

- 실제 이름, 닉네임, 위치, 직장, 학교, 커뮤니티명은 기록하지 않습니다.
- 대화 내용은 그대로 옮기지 않고 분위기와 배운 점만 요약합니다.
- 누군가를 평가하기보다 내 방송에 적용할 수 있는 태도와 방식을 적습니다.
- 민감한 이야기가 떠오르면 구체적인 정보는 덜어내고 감정의 방향만 남깁니다.
- 기록은 포트폴리오처럼 꾸미기보다 조용한 라디오 노트처럼 유지합니다.

## 폴더 구조

```text
logs/
  live/        # 내 라이브 방송 후 기록
  visit/       # 다른 방송 탐방 후 기록
templates/    # 반복해서 쓰는 기록 템플릿
notes/        # 아이디어와 안전 규칙
```

## 사용 방법

라이브 방송을 마친 뒤에는 `templates/live.md`를 복사해 `logs/live/`에 날짜별 파일로 저장합니다. 다른 방송을 들으며 배운 점이 있을 때는 `templates/visit.md`를 복사해 `logs/visit/`에 남깁니다. 한 주가 끝날 때는 `templates/weekly-review.md`로 방송 횟수와 작은 변화를 돌아봅니다.

스푼을 처음 익히며 알아둘 내용은 `notes/beginner-guide.md`에, 방송 중 들은 은어는 `notes/spoon-slang.md`에 따로 정리합니다. 익명 기록의 안전 기준은 `notes/safety-rules.md`를 기준으로 합니다.

피아노 방송에서 추천받은 곡과 연습 과정은 `practice/piano/`에 곡별로 정리합니다. 악보 원본, MR, 전체 연주 녹음 파일은 공개 저장소에 올리지 않습니다.

좋은 기록은 길지 않아도 됩니다. 오늘의 공기, 목소리의 속도, 대화가 편해진 순간, 다음에 한번 더 시도해보고 싶은 것만 남겨도 충분합니다.
